# Nino Nakano Theme (BetterDiscord)

Tema personalizado para BetterDiscord inspirado en **Nino Nakano** de *Gotoubun no Hanayome*, basado en **NotAnotherAnimeTheme v3**.

## Características

- Paleta rosa/magenta elegante y consistente.
- Fondo personalizado con transparencias oscuras.
- Acentos visuales en:
  - menciones
  - canales seleccionados/hover
  - barra de nuevos mensajes
  - indicadores de voz (speaking)
- Scrollbar y badges de no leídos personalizados.
- Ajustes extra de pulido visual (transiciones, focus visible, glow en botones).

## Requisitos

- [BetterDiscord](https://betterdiscord.app/)
- Conexión a internet (el tema importa la base NAAT desde URL):
  - `https://puckzxz.github.io/NotAnotherAnimeTheme/build/v3/naat.v3.css`

## Instalación

1. Descarga o clona este repositorio.
2. Copia `NinoNakano.theme.css` a la carpeta de temas de BetterDiscord:
   - **Windows:** `%appdata%/BetterDiscord/themes`
   - **macOS:** `~/Library/Application Support/BetterDiscord/themes`
   - **Linux:** `~/.config/BetterDiscord/themes`
3. Abre Discord → **Settings** → **BetterDiscord** → **Themes**.
4. Activa **NinoNakano.theme.css**.

## Personalización rápida

La mayoría de ajustes se controlan en `:root` dentro del archivo `NinoNakano.theme.css`.

Variables recomendadas para cambiar primero:

- `--theme-background-image`
- `--main-theme-color`
- `--theme-transparency`
- `--home-icon-image`
- `--font-size`
- `--server-listing-width`
